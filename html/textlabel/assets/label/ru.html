intelligent.museum

Spoken Language Identification: An AI-assisted Museum Label

2020/2021

Метаинформация
Автоадаптивная музейная этикетка на базе ИИ (искусственного интеллекта) с идентификацией языка

Программное обеспечение
Идентификация разговорной речи (машинное обучение: Tensorflow; набор речевых данных: Mozilla Common Voice)

Аппаратное обеспечение
Одноплатный компьютер, экран (10"), лазерный датчик, петличный микрофон.

Описание

Команда проекта intelligent.museum разработала прототип музейной этикетки с поддержкой ИИ для выставочного пространства. Он предназначен для идентификации родного языка исходя из разговорной речи посетителей выставки. С его помощью описание представленной работы и метаинформация автоматически переводятся на соответствующий язык.

Машинное распознавание разговорной речи не является тривиальной задачей. Чтобы научить машину безупречно распознавать предложения на определенном языке, необходим весь спектр данного языка. Для этого требуются невероятные объемы информации. Поскольку языки пересекаются по звучанию, чем больше языков нужно распознать, тем сложнее становится задача. Учитывая различные акценты, возраст, пол и другие характеристики, влияющие на речевой тракт, задача еще более усложняется. В ходе первых экспериментов прототип текстовой этикетки с поддержкой ИИ способен различать семь языков (немецкий, английский, французский, испанский, итальянский, китайский, русский) с общей точностью 83%.

До тех пор, пока этот прототип не станет масштабной технологией, используемой в международных музеях, необходимы дальнейшая разработка и внедрение решений. Это единственный способ обеспечить надежность работы этикетки в выставочном пространстве.
